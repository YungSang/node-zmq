# node-zmq

ØMQ module for Node.js

## Dependencies

 * [Node.js][node.js] - tested with v0.2.4
 * [ØMQ][zmq] - tested with [v2.1 (master 2010/11/22)][zeromq2]

## To Build & Install

	$ node-waf configure build

	$ node-waf install

## Examples

All examples follow [the ØMQ Guide][zguide]. Please refer to the guide.
