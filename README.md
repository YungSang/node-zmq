# node-zmq

ØMQ module for Node.js

## Dependencies

 * [Node.js][node.js] - tested with v2.0.3 and v2.0.4
 * [ØMQ][zmq] - tested with v2.0.10

## To Build & Install

	$ node-waf configure build

	$ node-waf install

## Examples

All examples follow [the ØMQ Guide][zguide]. Please refer to the guide.
